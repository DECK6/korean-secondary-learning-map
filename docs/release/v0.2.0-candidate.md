# v0.2.0-candidate 릴리스 보고서

릴리스 기준일: 2026-07-11

## 판정

이 산출물은 클래스·속성·제약을 정의한 OWL/RDFS/SKOS TBox, 정규화 데이터에서 생성한 JSON-LD/Turtle ABox, SHACL 형상, SPARQL 역량 질문을 함께 갖추므로 기술적으로 **형식 온톨로지**라고 할 수 있다. 다만 교육 내용의 사람 검토와 문서 재사용 권리 검토가 끝나지 않아 상태는 `candidate`, 권리는 `HOLD`다.

## 포함 범위

- 현행 총론·중학교·고등학교·교과·선택·전문교과·창체 공식 PDF 38개 receipt
- 중학교 13개 교과군, 24개 과목, 149개 영역, 714개 성취기준·주제
- 고등학교 34개 교과군, 759개 과목, 5,169개 영역, 50,749개 성취기준·주제
- 고등학교 학점 규칙 3개, 과목 관계 후보 473개, 선택 비교 묶음 33개, 예시 경로 33개
- 중학교→고등학교 과정 수준 전이 후보 398개(`continues` 6, `deepens` 174, `prepares-for` 218)
- 166,880개 RDF 노드, 1,798,025개 트리플, SCQ-01~20, 적대 탐지 10종
- 783개 과목을 지연 로딩하는 통합 탐색 UI

## v0.1 감사 후 수정

- 과목 식별자를 학교급·교과군·과목 범주·과목명 기준으로 다시 구성해 코드 접두부 차이로 생긴 중복을 제거했다.
- Unicode 대시를 정규화하고 중학교 선택 교과명·전문교과 범주 판별을 보완했다.
- 5,318개 `Domain`을 독립 노드로 실체화하고 성취기준·주제·클러스터 참조 무결성을 강제했다.
- 코드 배열 순서로 만든 자동 선수 관계 50,565건을 모두 제거했다. 현재 `LearningRelation`은 0건이다.
- 성취기준 요약을 `mechanical-derivative`로 명시하고 자동 요약의 어미 결함을 교정했다.
- 중→고 전이를 주제 단위 사실이 아니라 같은 교과군에 대한 과정 수준 후보로 낮추고 근거를 명시했다.
- RDF 변환에서 영역, 클러스터, 관계 한정자, 경로 단계, 출처·권리·검토 메타데이터가 소실되지 않도록 확장했다.
- `CourseRelation`이 `TransitionAlignment`로 잘못 추론되던 속성 domain 충돌을 제거했다.
- SHACL을 전체 ABox에 실행하고 SPARQL 20개를 실제 실행 결과와 대조하도록 검증기를 강화했다.

## 공개 전 남은 사람 검토

기계적 요약·세부 주제·증거·평가 질문·과목 관계·전이·경로는 공식 교육과정 사실과 구분된 후보이다. 교과 및 직업계 전문가가 이 해석을 검토하고, 문서별 재사용 조건이 확인되기 전에는 `candidate`와 `RIGHTS HOLD`를 해제하지 않는다.
