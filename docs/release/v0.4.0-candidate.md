# v0.4.0-candidate 릴리스 보고서

릴리스 기준일: 2026-07-13

## 변경 목적

v0.3은 중학교 수학 일부를 제외하면 주제 관계가 비어 있었고, 공식 문서에서 기계적으로 만든 광범위한 과목·학교급 후보가 재빌드 때마다 검토 산출물을 덮어쓸 수 있었다. v0.4는 중학교와 고등학교 전 과목의 학습 그래프를 결정적으로 재생성하고, 공식 근거 관계와 비강제 탐색 순서를 명시적으로 분리한다.

## 재현 가능한 관계 빌드

- `scripts/lib/reviewed-learning-relation-specs.mjs`에 공식 직접 연결을 사람이 검토 가능한 작은 사양으로 고정했다.
- `scripts/build-learning-relations.mjs`가 전 과목 탐색 관계, 과목 연계와 bridge를 stable ID로 생성한다.
- `build`, `rebuild:all`, `check`, `verify`에 관계 빌드 또는 결정성 검사를 연결했다.
- 공식 PDF 38개를 다시 내려받은 전체 원천 재빌드에서 중학교 714개·고등학교 50,749개 성취기준과 추출 진단 0건을 재현했다.

## 관계 의미 경계

- 공식 내용 체계·해설이 직접 뒷받침하는 경우에만 `required-prerequisite`를 사용한다.
- 공식 문서 배열은 `recommended-before` 탐색 순서이며 공식 이수 조건이 아니다.
- 교과 총론의 직접 연계 설명은 `official-source`와 페이지 locator를 가진 `reviewed-recommendation`으로 기록한다.
- 과정 수준 중→고 연결은 `repository-authored`이며 입학·진로 적합성 판정을 뜻하지 않는다.
