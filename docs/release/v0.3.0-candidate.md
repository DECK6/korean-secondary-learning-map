# v0.3.0-candidate 릴리스 보고서

릴리스 기준일: 2026-07-11

## 변경 목적

v0.2 집계는 중학교 714개 성취기준을 714개 주제로만 표현해 초등 `kr-full-depth-v0.4`의 620개 기준→1,956개 주제보다 학습지도 깊이가 얕았다. 또한 고등학교 총계에 528개 직업계 전문교과와 47,625개 세분화 성취기준을 합쳐 학교급 비교를 왜곡했다.

## 중학교 주제 분해

- 공식 중학교 과목·영역·성취기준 수는 24·149·714로 유지했다.
- 기존 714개 주제 ID를 `standard-core`/`core`로 보존했다.
- 과목별 `subject-facet` 주제 1,446개를 추가해 총 2,160개로 확장했다.
- 기준당 주제는 기술·가정/정보 2개, 기타 3개, 국어 4개, 도덕 5개다.
- 분포는 2개 77기준, 3개 564기준, 4개 51기준, 5개 22기준이며 평균은 3.03개다.
- 추가 주제는 `middle-subject-facet-decomposition-v1`, 분해 종류, facet key, 정렬 역할, 관찰 증거, 평가 질문을 보존한다.
- 수량을 맞추기 위한 선수 관계는 만들지 않아 `LearningRelation`은 계속 0건이다.

## 비교 가능한 집계

| 범위 | 과목 | 영역 | 성취기준 | 주제 |
| --- | ---: | ---: | ---: | ---: |
| 중학교 | 24 | 149 | 714 | 2,160 |

초등은 기준당 주제 3.15개, 중학교는 3.03개로 분해 밀도가 비슷해졌다. 초등의 1,894개 선수 추천은 해당 저장소 모델의 후보 관계이며, 중학교에 복사하지 않는다.

## 형식 모델과 UI

- `MiddleLearningTopic` 하위 클래스를 추가했다.
- JSON Schema·RDF·JSON-LD context·SHACL에서 분해 종류와 facet key를 검증한다.
- 고등학교를 비직업계와 직업계 전문교과로 분리 집계하고 UI 필터를 추가했다.
- 과목 상세에서 성취기준별 복수 주제·증거·평가 질문을 모두 표시한다.
- 최종 ABox는 168,326개 리소스와 1,879,104개 트리플이다.
