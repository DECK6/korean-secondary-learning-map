## 채택 관계

### 중학교 내용 체계

채택 관계 없음. 중학교 내용 요소와 성취기준이 1~3학년 단일 학년군으로 묶여 있어 중학교 내부의 학년별 선후를 확인할 수 없다.

### 중학교 성취기준 해설·적용 시 고려 사항

채택 관계 없음. 원리와 방법의 활용 관계는 설명하지만, 중학교의 특정 성취기준을 이미 학습한 선수 내용으로 명시하지 않는다.

### 초등학교 해설에서 중학교

## 제외 판단

- 중학교 1~3학년군 안의 배열 순서나 일반적인 난이도는 선수 관계로 사용하지 않았다.
- 초등 3~4학년군은 같은 갈래의 5~6학년군 요소가 직접 선행하므로 중학교로 건너뛰는 중복 간선을 만들지 않았다.
- 중학교 적용 시 고려 사항의 영역 단위 초등 연계 문장은 내용 체계표에서 확인되지 않는 별도 특정 코드 관계로 확장하지 않았다.
- 고교 설계 개요의 전체 과목 수준 연계만으로는 간선을 만들지 않고, 각 과목의 성격 문장과 직접 대응하는 성취기준을 함께 확인했다.
- 스포츠 과학은 과목 성격에서 중학교 경기 기능·방법·전략을 사회과학 및 자연과학 원리 적용의 공통 기반으로 명시하므로 두 과학 영역의 적용 성취기준에 각각 연결했다.

## 검토 한계

이 결과는 저장소 내부의 공식 문서 근거 검토이며 체육 교과 전문가의 승인 결과가 아니다. 내용 체계표의 학년군 계열과 고교 과목 성격의 명시적 연계를 데이터 간선으로 옮긴 것으로, 학교의 실제 편성 순서나 학습자별 선수 능력을 규정하지 않는다.
