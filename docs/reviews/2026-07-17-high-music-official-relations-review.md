# 고등학교 음악과 공식 선수학습 관계 검토 (2026-07-17)

- 원천: 교육부 고시 제2022-33호 [별책12] 음악과 교육과정
- annexId: `kr-nec-2024-3-annex12`
- 범위: 고등학교 4과목 37개 성취기준(음악 13개, 음악 연주와 창작 8개, 음악 감상과 비평 8개, 음악과 미디어 8개)
- 방법: 고교 과목 설계, 내용 체계, 성취기준 해설, 적용 시 고려 사항에서 특정 성취기준 사이의 선행 학습을 직접 지목한 경우만 인정했다. 목차와 성취기준 배열 순서는 근거로 사용하지 않았다.
- 식별자 검증: 대상 코드가 `data/kr/high/standards.json`에 존재하고 별책12를 출처로 참조하는지 대조했다.

## 고등학교 내용 체계·과목 설계 기반 관계 (0건)

내용 체계는 과목별 영역과 요소를 제시하지만, 37개 성취기준 사이의 이수 순서나 선수 기준을 직접 지정하지 않는다. p.29–31의 선택 과목 세분화·확장 설명과 영역 1·2 구성 방향, p.53과 p.65의 일반 선택 <음악> 심화 설명은 과목·영역 수준 관계이며, 개별 성취기준 대응 근거가 없어 수록하지 않았다.

## 고등학교 해설·적용 시 고려 사항 기반 관계 (0건)

해설의 ‘바탕으로’와 유사한 표현은 각 성취기준 안에서 이해·탐색 뒤 표현·토론으로 이어지는 활동 절차를 설명할 뿐, 별도의 선수 성취기준을 지목하지 않는다. p.81의 미디어 탐구 기초 설명도 특정 후속 성취기준을 지정하지 않아 관계로 만들지 않았다.

## 학교급 간 관계 (0건)

`middleRequired`, `middleCommentaryRequired`, `elementaryToMiddleRequired`, `elementaryCommentaryRequired`는 고교 모듈 범위 밖이므로 비워 두었다. p.39와 p.40에는 중학교 학습 내용의 심화가 언급되지만, 별도 중→고 작업과 중복하지 않도록 `middleToHighRequired`도 지시대로 비워 두었다.

## 검토 한계

이 문서는 공식 별책의 명시 문구만을 판정한 저장소 내부 검토 결과이며 음악 교과 전문가의 승인 기록이 아니다. 과목 수준의 심화·연계 설명, 내용 요소의 유사성, 문서 배열 순서를 성취기준 관계로 추론하지 않았다. 따라서 공식 문서에 코드 수준의 선후 근거가 추가로 확인되기 전까지 빈 배열을 유지한다.
