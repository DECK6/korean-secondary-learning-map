# 식품·조리 전문교과 공식 선수학습 관계 검토 (2026-07-17)

- 원천: 교육부 고시 제2022-33호 [별책 28] 식품·조리 전문 교과 교육과정(국가교육위원회 고시 제2024-3호 일부개정 포함)
- annexId: `kr-nec-2024-3-annex28`
- 방법: 내용 체계, 과목 설계, 성취기준 적용 시 고려사항, 과목 성격의 명시적 선행 표현을 전수 검색하고 성취기준 코드 수준의 방향과 필수성이 확인되는 경우만 선별했다.

## 채굴 결과

- `highRequired`: 0건. 내용 체계와 과목 설계는 영역·능력단위를 제시하지만 성취기준 사이의 선후를 직접 규정하지 않는다.
- `highCommentaryRequired`: 0건. 코드·과목 간 연계 안내는 다수 있으나 학습 순서나 필수 선행을 특정 코드 수준에서 명시하지 않는다.
- `middleRequired`: 0건.
- `middleCommentaryRequired`: 0건.
- `elementaryToMiddleRequired`: 0건.
- `elementaryCommentaryRequired`: 0건.
- `middleToHighRequired`: 0건. 중학교에서 고등학교로의 관계는 지시에 따라 채굴하지 않았다.

## 공식 근거 관계

해당 없음.

## 제외 판단

- “연계하여 지도한다”는 표현은 학습의 방향이나 필수 선행을 정하지 않으므로 선수관계로 해석하지 않았다.
- 전공 일반 과목이 전공 실무 과목의 선수 과목이라는 교과 수준 안내는 개별 성취기준 대응을 특정하지 않아 제외했다.
- NCS 능력단위와 성취기준의 나열 순서, 기초·심화 내용의 묶음은 특정 코드 쌍의 선후 근거로 사용하지 않았다.
- 기존 과목 수준 연계를 성취기준 수준 근거 없이 복제하지 않았다.

## 검토 한계

이 결과는 별책 텍스트에 명시된 코드 수준 선수관계만 보수적으로 수록한 것이다. 수업 운영상 타당한 연계나 과목 수준의 위계가 있더라도 공식 문장이 특정 성취기준 쌍의 선후를 직접 뒷받침하지 않으면 포함하지 않았다.
