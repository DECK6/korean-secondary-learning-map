# 2026-07-13 중·고 전 교과 학습 관계 내부 검토

## 검토 범위

- 기준선: 현행 2022 개정 교육과정 공식 PDF 38개
- 중학교: 24과목, 714성취기준, 2,160토픽, 2,155관계
- 고등학교: 759과목, 50,749성취기준·토픽, 50,029관계
- 과목 관계: 공식 교과 설계의 연계·심화 설명 39건
- 학교급 bridge: 중→고 50건(과정 수준 37, 수학 주제 수준 13), 초등 수학→중학교 수학 19건
- 검토 상태: 저장소 소유자 승인 범위의 `internal-reviewed`; 교육부·교과 전문가·학교 현장 승인이 아님

## 관계 층위

1. `required-prerequisite`: 별책8 수학의 내용 체계표 또는 성취기준 해설이 직접 뒷받침하는 중학교 30건과 초→중 19건에만 사용했다.
2. `official-source` 추천: 국어·수학·과학·기술가정·영어·체육 교과 설계가 직접 설명한 고등학교 과목 연계 39건과 수학 중→고 주제 연계 13건이다.
3. `repository-authored` 추천: 공식 PDF의 과목·영역·성취기준 배열을 전 과목 탐색용 순서로 검토한 관계다. 엄격한 이수 조건이나 교육적 필수 선수를 주장하지 않는다.
4. 과정 수준 bridge: 중학교 24개 전 과목을 대응하는 고등학교 기초 과목에 연결한다. 입학·수강 요건을 뜻하지 않는다.

## 재현

```bash
bun run rebuild:all
bun run check:relations
bun run verify
```

`build:data` 직후의 관계 0건·광범위 후보 과목 연결을 `build:relations`가 검토된 결정적 산출물로 교체한다. `--check` 모드는 현재 JSON과 재계산 결과가 한 바이트라도 다르면 실패한다.

## 남은 한계

전 과목을 탐색할 수 있는 그래프는 완성했지만, 비강제 배열 관계를 교과별 의미 선수 관계로 더 세분화하는 일은 교과·직업계 전문가의 후속 검토 대상이다. 공식 원문 비포함과 문서별 권리 `HOLD`도 유지한다.
