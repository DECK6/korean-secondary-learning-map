# 전기·전자 전문교과 공식 선수학습 관계 검토 (2026-07-17)

- 원천: 국가교육위원회 고시 제2024-3호 일부개정 [별책 34] 전기·전자 전문 교과 교육과정
- annexId: `kr-nec-2024-3-annex34`
- 방법: 성취기준 적용 시 고려 사항, 과목 성격, 교수·학습 서술에서 선수와 후속 성취기준 코드를 함께 특정하고 학습 순서를 직접 밝히는지 확인했다.
- 결과: `highRequired` 0건, `highCommentaryRequired` 0건

## 채굴 결과

채택한 성취기준 수준 관계가 없다.

## 빈 배열과 제외 판단

- `highRequired`는 0건이다. 내용 체계표와 NCS 능력단위 목록은 학습 내용을 분류하지만 코드 간 선수 관계를 직접 표시하지 않는다.
- `highCommentaryRequired`는 0건이다. 전기 회로·전자 회로·디지털 논리 회로를 실무 과목의 선수 과목으로 설명한 문장은 후속 성취기준을 특정하지 않는다.
- 자동화 설비 응용에서 이전 학습 영역을 언급한 문구와 의료 기기 품질 관리 시스템의 선행 학습 확인 문구도 출발·도착 코드를 직접 특정하지 않아 제외했다.
- ‘관련 교과 및 선수 학습 과목과 연계’ 같은 포괄적 안내와 단순 연계 문구는 방향성 있는 코드 관계로 사용하지 않았다.
- 나머지 학교급 배열은 고등학교 전용 모듈이어서 비워 두었고, `middleToHighRequired`는 별도 작업과의 중복을 막기 위해 비워 두었다.

## 검토 한계

- 이 검토는 공식 별책 텍스트와 저장소 성취기준 코드의 내부 대조 결과이며 교과 전문가의 승인 기록이 아니다.
- 작업 공정상 자연스러운 순서나 성취기준 나열 순서는 공식 선수학습 근거로 간주하지 않았다.
- 관계가 0건인 것은 채굴 누락 수를 뜻하지 않으며, 코드 수준의 명시 근거가 없을 때 빈 배열을 유지한다는 원칙을 적용한 결과다.
