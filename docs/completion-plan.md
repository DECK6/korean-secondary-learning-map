# 완성 계획과 수용 기준

## 완성 정의

이 프로젝트의 “완성”은 다음 산출물이 모두 존재하고 자동 게이트를 통과한 상태다.

1. 현행 2022 개정 교육과정의 중학교·고등학교 공식 과목과 성취기준 코드 인벤토리
2. 중학교와 고등학교의 독립 데이터 릴리스 및 중→고 전이 bridge 릴리스
3. 성취기준별 세부 학습 주제, 관찰 가능한 증거, 평가 질문, 근거가 명시된 검토 가능한 관계 주장
4. 공통·선택·전문 교과와 고등학교 학점·선택 구조를 구분하는 데이터 모델
5. OWL/RDFS/SKOS TBox, JSON-LD/Turtle ABox, SHACL, SPARQL 역량 질문과 적대 fixture
6. 중학교·고등학교·전이·과목 비교·경로 탐색을 제공하는 통합 UI
7. 결정적 빌드, manifest SHA-256, CI, 콘텐츠·참조·권리·브라우저 검증
8. 출처·권리·검토 한계와 재현 명령을 설명하는 릴리스 문서

자동 검증 통과는 교육부·국가교육위원회·NCIC 승인이나 교과 전문가의 포괄적 승인, 공식 원문 재사용 허가를 뜻하지 않는다.

## 공식 원문 기준선

- 2026년 국가교육위원회 고시 제2026-1호: 총론, 중학교, 고등학교의 최신 개정본
- 2024년 국가교육위원회 고시 제2024-3호: 해당 고시에서 다시 발행하거나 개정한 교과·선택·전문교과
- 2022년 교육부 고시 제2022-33호: 이후 고시가 대체하지 않은 교과·선택·창체 별책

각 파일은 `sources/official/source-catalog.json`의 선택 근거와 `source-receipts.json`의 실제 해시로 고정한다. 최신 고시가 바뀌면 기존 릴리스를 조용히 덮지 않고 새 데이터 릴리스와 bridge 재검토를 시작한다.
