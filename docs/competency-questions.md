## 적대 fixture 최소 세트

- 고등학교 과목에 근거 없이 `grade=10`을 강제한 레코드
- 국가 수준 `Course`를 실제 학교 `CourseOffering`으로 선언한 레코드
- 모델 생성 선수 후보를 `official-prerequisite`로 승격한 레코드
- 선택 과목 범주가 없거나 서로 충돌하는 과목
- 과목·성취기준 코드 locator가 없는 `official-source-checked` 레코드
- 주제 양 끝점만 같고 적용 범위가 다른 두 주장을 하나의 ID로 충돌시킨 레코드
- 추천 경로에서 `notOfficialRequirement` 경계가 사라진 레코드
- 초등 기존 IRI를 새 중등 ID 규칙으로 다시 발급한 레코드
- 권리 `HOLD`를 형식 검증 통과로 자동 해제한 레코드
- 순환을 허용하지 않는 관계 종류에서 생긴 사이클

## 릴리스 불변식

- 모든 성취기준은 정확히 하나의 공식 과목에 속한다.
- 모든 학습 주제는 하나 이상의 성취기준 정렬 또는 명시적 비정렬 사유를 가진다.
- 모든 확정 학습 관계는 이유·근거·검토 상태를 가진다.
- 공식 필수 선수 관계는 공식 출처 없이는 생성할 수 없다.
- 모든 과목 범주는 통제 어휘에 존재해야 한다.
- `CourseOffering`이 없는 과목은 “교육과정 정의됨”으로만 표시된다.
- 파생 관계는 파생 규칙과 경로를 재현할 수 있어야 한다.
- 전체 출력은 동일 입력에서 바이트 단위로 재현 가능해야 한다.
